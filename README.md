# 📄 중간고사 실기 시험 – 오픈북 (약 30분)

## 🧠 목표

작은 학급을 위한 성적 계산기 프로그램을 작성하세요.

프로그램은 사용자로부터 학생 정보를 입력받아 평균 점수를 계산하고 출력해야 합니다.

## 프로그램 요구사항

1. 학생 수를 입력받습니다.
2. 각 학생에 대해:
   - 이름 입력 (`string`)
   - 점수 3개 입력 (`vector<int>`)
3. 평균 점수를 계산합니다.
4. 모든 학생의 이름과 평균 점수를 출력합니다.

## 필수 조건

- 학생 이름은 string으로 저장
- 점수는 vector<int>로 저장
- 학생 정보는 구조체(struct) 또는 여러 개의 배열/vector를 사용하여 저장
- 평균 계산 함수 1개 이상 작성
- 프로그램은 반드시 다음 세 파일로 작성:
   - `main.cpp`
   - `functions.cpp`
   - `functions.h`
- 결과는 입력한 순서대로 출력

## 디버깅 문제 (먼저 수정 후 사용)

다음 코드는 평균을 계산하는 함수입니다.

코드를 읽고 오류를 수정하세요.

```c
double calculateAverage(vector<int> grades) {
    int total = 0

    for (int i = 0; i < grads.size(); i++) {
        total = grades[i];
    }

    return total / grades.size();
}
```

### 해야 할 일

- 철자 및 구문 오류 수정 (2개)
- 점수 합계가 올바르게 계산되도록 수정
- 평균이 소수로 나오도록 수정
- 수정한 함수를 프로그램에서 사용

## 추가 점수 (+보너스)

다음 기능 중 하나 이상을 구현하면 추가 점수를 받을 수 있습니다.

- 이름순 정렬
- 최고 평균 점수 학생 출력
- 90점 이상 학생에게 `"Excellent"` 출력
- 숫자가 아닌 값 입력 시 오류 메시지 출력

## 출력 예시

```bash
Enter number of students: 2

== Student 1 ==
Enter name: Alice
Enter 3 scores: 90 85 100

== Student 2 ==
Enter name: Bob
Enter 3 scores: 70 75 60

--- Grade Summary ---

Alice: 91.7
Bob: 68.3
```
